
# Тестовое задание для вакансии в Neuromap

Описание <https://github.com/ivryb/test/blob/master/meetings-light.md>

## Используется

- Vue
- Vuex
- Vue Router
- Moment.js
- Element UI
- Webpack

## Запуск приложения

- dist/index.html

``` bash
# Установка зависимостей
npm install

# Запуск в режиме development
npm run dev

# Сборка для  production
npm run build

```
